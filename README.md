# Cloud-Native Weather Notification Engine

A Python-based application that fetches real-time weather data for Jaipur via APIs. This project demonstrates API integration and Docker containerization.

## Features
* Real-time temperature and wind speed tracking.
* Dockerized for consistent deployment across environments.
* Professional error handling for API requests.

## How to Run
1. *Build:* docker build -t weather-notification-app .
2. *Run:* docker run weather-notification-app
