# Cloud-Native Weather Notification Engine

## Description
A Python-based application that fetches real-time weather data via APIs. This project is part of my BCA portfolio to demonstrate API integration and DevOps practices.

## Features
* Real-time temperature and wind speed tracking.
* Virtual environment isolation for dependency management.
* Professional error handling for API requests.

## Setup
1. Clone the repository.
2. Create a virtual environment: python -m venv venv.
3. Install dependencies: pip install requests.
